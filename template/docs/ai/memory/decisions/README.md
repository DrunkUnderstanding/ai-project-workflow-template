# Decision Records

Use this directory for important project decisions that should not be lost in chat history.

Decision records are for choices that affect future work, such as architecture, data format, asset pipeline, framework adoption, gameplay direction, or validation strategy.

## Naming

Use this format:

```text
NNNN-short-title.md
```

Examples:

```text
0001-use-scriptableobjects-for-prototype-config.md
0002-defer-addressables-until-mvp.md
```

## Template

```markdown
# Decision: Title

## Status

Proposed | Accepted | Rejected | Superseded

## Context

What situation or problem led to this decision?

## Decision

What did we choose?

## Consequences

What becomes easier, harder, or constrained because of this?

## Alternatives Considered

What other options were considered?

## Date

YYYY-MM-DD
```
