# AI Asset Pipeline

> Read this before generating or importing AI-created art, audio, icons, or other media.

## Current Decision

AI-generated draft assets start in `AI_Assets/` and only move into `Assets/` after review.

## Rules

- Use `AI_Assets/` for drafts, references, intermediate files, and rejected variants.
- Use production `Assets/` folders only for accepted assets.
- Keep prompts, source references, and processing notes near generated assets when useful.
- Do not overwrite existing production art without explicit approval.

## Verification

- Check dimensions, transparency, naming, and import suitability before moving into `Assets/`.
- Confirm Unity imports accepted assets correctly.
