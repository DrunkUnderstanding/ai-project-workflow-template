# 经验迁移指南

这个模板的价值在于：项目经验可以逐步回流，变成下一个项目的起点。

## 适合迁回模板的内容

- 通用 AI 工作流
- 人类确认门
- spec 模板
- 验证流程
- 架构经验教训
- 决策记录格式
- 跨项目 skill 约定

## 不适合迁回模板的内容

- 私有业务逻辑
- 某个游戏的具体玩法
- 内部服务地址
- 密钥、token、账号
- 特定项目路径
- 临时生成资源

## 迁移步骤

1. 去掉项目名和专用路径。
2. 用占位符替换项目专有内容。
3. 通用内容放 `template/`。
4. 引擎或行业专用内容放 `examples/`。
5. 更新 README 和使用说明。
6. 提交并打版本。
